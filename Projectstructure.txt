# Confluence Helper Project Structure

```
confluencehelper/
│
├── app.py                    # Main Flask application
├── config.yaml              # Configuration (add to .gitignore)
├── requirements.txt         # Python dependencies
├── .gitignore              # Git ignore file
│
├── core/                    # Core business logic
│   ├── __init__.py
│   ├── analyzer.py         # ClusterAnalyzer class
│   └── suggester.py        # SuggestionEngine class
│
├── integrations/           # External service integrations
│   ├── __init__.py
│   ├── confluence_client.py   # ConfluenceClient class
│   └── db_manager.py          # DatabaseManager class
│
├── templates/              # HTML templates
│   ├── base.html          # Base template with common layout
│   ├── index.html         # Space listing page
│   ├── space.html         # Cluster listing page
│   ├── cluster.html       # Cluster details & suggestions
│   └── error.html         # Error page
│
├── static/                 # Static files
│   ├── css/
│   │   └── style.css      # Custom styles
│   └── js/
│       └── main.js        # JavaScript for interactivity
│
└── tests/                  # Test files
    ├── __init__.py
    ├── test_analyzer.py
    ├── test_confluence_client.py
    ├── test_db_manager.py
    └── test_suggester.py
```
